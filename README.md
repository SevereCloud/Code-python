## Сортировка вставками. Сортировка слиянием. Анализ алгоритмов сортировки.
### Сортировка вставками
![тут должна быть картинка](https://upload.wikimedia.org/wikipedia/commons/0/0f/Insertion-sort-example-300px.gif)
```python
for i in range(len(array)):
    x = array[i]
    m = i
    while (array[m-1] > x) & (m > 0):
        array[m] = array[m-1]
        m -= 1
    array[m] = x
```
Мы имеем два цикла, которые повторяются n раз, значит время в худшем случае O(n) = n * n = **n²**

### Сортировка слиянием

```python
def mergesort(array):
    if len(array) > 1:
        mid = int(len(array)/2)
        lefthalf = array[:mid]
        righthalf = array[mid:]

        mergesort(lefthalf)
        mergesort(righthalf)

        i = 0
        j = 0
        k = 0
        while i < len(lefthalf) and j < len(righthalf):
            if lefthalf[i] < righthalf[j]:
                array[k] = lefthalf[i]
                i = i + 1
            else:
                array[k] = righthalf[j]
                j = j + 1
            k = k + 1

        while i < len(lefthalf):
            array[k] = lefthalf[i]
            i = i+1
            k = k+1

        while j < len(righthalf):
            array[k] = righthalf[j]
            j = j + 1
            k = k + 1
    return array      
```
